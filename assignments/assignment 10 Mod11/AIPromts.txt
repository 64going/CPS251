Your code is **almost correct**, but the error comes from this line:

```kotlin
supportingText = { if (isTitleEmpty) { Text("Title cannot be empty") } }
```

In **Material 3** (`OutlinedTextField`), the lambda passed to `supportingText` must **always return a composable**, not just conditionally.
When the `if` statement doesn‚Äôt run, the lambda returns **Unit**, which is **not a composable**, so Compose throws an error.

---

## ‚úÖ Fix: Provide an `else` with an empty composable

```kotlin
supportingText = {
    if (isTitleEmpty) {
        Text("Title cannot be empty")
    } else {
        Spacer(Modifier.height(0.dp))   // or nothing composable
    }
}
```

A cleaner empty composable:

```kotlin
supportingText = {
    if (isTitleEmpty) Text("Title cannot be empty")
}
```

‚û°Ô∏è This version works only if your Compose version allows conditional returning inside the lambda.
If yours does **not**, use a safe empty composable:

```kotlin
supportingText = {
    if (isTitleEmpty) {
        Text("Title cannot be empty")
    } else {
        Box {}
    }
}
```

A safer form is:

onValueChange = {

# ‚úÖ **Root Cause**

You probably have this import:

```kotlin
import androidx.compose.material.OutlinedTextField   // ‚ùå Material 2
```

But `supportingText` and `OutlinedTextFieldDefaults.colors()` belong to:

```kotlin
import androidx.compose.material3.OutlinedTextField   // ‚úÖ Material 3
import androidx.compose.material3.OutlinedTextFieldDefaults
import androidx.compose.material3.Text
```

Material 2 `OutlinedTextField` ‚Üí does **not** have:

* `supportingText`
* `OutlinedTextFieldDefaults.colors()`

So the compiler cannot match your parameters and prints the giant overload error.

---

# ‚úÖ **Fix: Use Material 3 imports**

Make sure your imports look like this:

```kotlin
import androidx.compose.material3.OutlinedTextField
import androidx.compose.material3.OutlinedTextFieldDefaults
import androidx.compose.material3.Text
import androidx.compose.material3.MaterialTheme
```

And remove this if it's present:

```kotlin
import androidx.compose.material.*   // ‚ùå
```
